# 海天气象V2.1.3全新上线

海天气象（https://cornicelli.net/）是一个致力于为涉气象领域科学研究和技术应用提供统一规范、精准精细、高质高效的气象数据资源支持的数据信息服务平台，着力解决气象数据资源获取、存储、开发、应用等环节的一系列瓶颈难题，帮助气象数据用户将数据获取难度降至0从而将更多精力用于气象数据的价值挖掘和开发创造。
目前海天气象已发布全国市区县天气预报数据集、全国市区县历史天气数据集、气象决策服务可视化图表、城市地理信息数据库、气象数据Python SDK等数据资源和配套支持软件。用户可以通过海天气象数据信息服务平台以极简优雅的方式一站式获取丰富的气象数据资源。

## 一、功能介绍

海天气象数据信息服务平台具有完善的气象数据资源和简单快捷的数据处理使用逻辑，力争用一行代码满足用户的所有数据需求，除了丰富气象数据资源，海天气象还提供与之配套的完整开发工具，全部由Python语言实现，同时继承Python语言优雅简便的设计原则，给用户最好的气象数据使用体验。除此之外还有详尽的数据开发文档以及全套示例源代码，均可通过网页快速访问，随时解决用户数据使用过程中的难题。
截至当前V2.1.3版本，海天气象主要提供以下7种功能：

- 气象数据图表浏览
- 气象数据下载
- 气象数据API调取-
- 气象数据集成开发工具（Python SDK）
- 气象数据开发文档和示例源代码
- 气象决策报表下载
- 在线论坛和常见问题

### 1. 气象数据图表浏览

提供全国市区县未来10日逐小时天气预报信息和全国市区县2000~2025年历史天气信息在线查看功能，共包含40+种气象要素，用户可通过动态交互式图表浏览全国3191个市区县的未来240小时天气预报详情和过去26年的历史天气详情，提供的图表包括气温变化折线图、气压变化折线图、降水量柱形图、相对湿度折线图、10米和100米风速折线图、各类辐射折线图等，可通过点选、放缩功能浏览全国所有城市、市辖区区和县的天气预报数据和历史天气数据。

### 2. 气象数据下载

提供全国市区县未来10日逐小时天气预报数据和2000~2025年逐小时历史天气数据下载功能，历史天气数据的逐日、逐月、逐年统计分析数据也可在线下载，用户选定城市和时间后即可下载数据，数据存储格式为CSV格式文本文件，Excel和文本阅读软件可以直接打开查看，也可使用Python等编程语言快速导入分析，也可直接导入MySQL数据库本地查询使用。更可以批量打包下载全国数据，一站式解决用户所有数据烦恼。针对天气预报数据提供自动拉取更新功能，用户可通过Python SDK实现本地气象数据与云端的自动批量同步更新。

### 3. 气象数据API调取

共提供7种API接口，解决轻量化开发场景中气象数据使用难题，用户能够在WEB应用或本地程序中通过构建url发送request请求的方式快速调取全站的天气预报数据和历史天气数据，满足各种开发场景和科学研究背景下的多样化气象数据调取需求，可以自定义时间跨度、气象要素种类和统计分析尺度迅速调取全国任意市区县或经纬度格点（仅限预报数据）的全部天气预报数据和历史天气数据，还提供经纬度地名查询接口，能够帮助用户迅速定位坐标地点和海拔。

### 4. 气象数据集成开发工具

提供Python SDK实现全站历史数据和预报数据的查询访问、批量下载、同步更新和账户管理功能，借助SDK用户通常只需一行代码获取实现调取任意地点、任意时间任意、气象要素的数据，既能下载数据到本地又能通过在线API查询，十分方便灵活，并且能够实现数据的批量下载和天气预报数据的自动更新功能，对于气象数据的统计分析也能轻松搞定，通过SDK所有气象要素均能迅速查询到对应日、月、年尺度的最大值、最小值、平均值、累积值（仅降水量）。

### 5. 气象数据开发文档和示例源代码

海天气象数据信息服务平台提供所有历史天气数据和天气预报数据的详细开发文档和示例源代码，使用户能够通过复制粘贴式编程迅速实现想要的功能，让所有的数据资源触手可及。开发文档可迅速查询所有的气象要素参数说明、定义和单位以及天气现象代码，示例源代码可以直接下载，并标注了详细的注释，可以帮助用户快速深入理解数据的存储组织结构和访问方法。

### 6. 气象决策报表下载

提供全国城市未来一周逐日全要素气象预报决策图表和省会城市未来一周气象预报详情分析图表，可以使用户快速查看关注地点的气温（包括体感温度、露点温度、地温等）、气压、相对湿度、云量、风速风向、降水量、天气现象变化趋势，通过的丰富的折线图、柱形图、饼图和风玫瑰图帮助决策用户深入理解未来天气动向和气象要素变化规律，从而给出更佳咨询建议和应对策略，深入契合交通、电力、农林渔牧、生态环境、资源开发、户外运动等行业的典型使用场景。

### 7. 在线论坛和常见问题

提供在线论坛供用户交流数据使用经验和解决数据处理难题，常见问题页面收录了开发者和用户经常遇到的使用难题，便于用户快速突破思维定势，寻找最佳的问题处理对策。

## 二、气象要素种类介绍

天气预报数据共有32种气象要素，历史天气共有40种气象要素，均包含在风、蒸发、云量、能见度、、降水、土壤湿度、气温、气压、动力指数、天气现象、辐射、大气湿度、低温等13个大类中，能够覆盖理论研究和实践应用中的大多数气象数据需求场景。

# 二、数据来源

使用的海况基础数据来自CMEMS的全球海浪再分析产品，其提供的海浪数据是目前全球海洋学研究、航运规划及气象预报中最权威的基准数据之一。时间分辨率3小时，空间分辨率0.25°。包括时间、VHM0、VMDR、VTM02等变量，其中VHM0为有效波高，VMDR为平均波向，VTM02为平均波周期。数据时间为1980年至今。

# 三、原理简介

波浪散布图的参数定义：有效波高VHM0（Significant Wave Height）周期为峰值周期。

波高等级划分标准：具体波高区间划分间隔为1米。

波高与周期的联合概率统计方法：波高-周期散布图分为2种，分别为全时段内统计分析和按月统计分析，示例散布图分别如下：

全时段内波高-周期散布图对所有时间范围内的波高、周期数据进行频数统计，波高步长为1米，周期步长为1秒，对应的单元格数字为该周期范围内波高数值出现的次数，并对每个周期和每个波高间隔进行了汇总统计，计算出总频数标注在表格右侧和下方。

各月波高-周期散布图对应的时间范围为月份，为1980年至今所有对应月份的波高和周期数据频数统计。

波高预测模型简介：

气候态历史均值预测法（Climatology Forecasting），统计分析1980至今共计47年的历史海况数据，用最邻近时间最邻近航点的期望波高预测航行时间的真实波高。

模型特性细节如下：

使用滑动时间窗口（前后3天）。通过对比目标日期前后3天（共7天）的历史同小时数据，样本量为322个，统计学上的期望值会大幅趋于稳定。

使用高分位数评估，防范极端海况（航海刚需），同时计算95%分位数。这代表“历史上95%的概率浪高不会超过这个数值”，为船长提供安全包络线。

使用反距离权重插值（IDW, Inverse Distance Weighting）。寻找最近的3个航点，按距离的平方倒数分配权重进行平滑计算，

数据内容：

599个航点1980至今3小时分辨率波高、周期、波向数据，格式为Parquet（类似于CSV）

图集：

599个航点全时段波高-周期散布图、逐月波高0周期散布图、全时段波浪玫瑰图、逐月波浪玫瑰图

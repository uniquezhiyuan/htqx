# 海天气象数据更新提示

海天气象数据信息服务平台（[https://htqx.cn/](https://htqx.cn/)）近期对天气预报数据和历史天气数据服务进行了优化升级，在保留原有服务的基础上分别对以上两项数据集进行了拓展，主要有两项重大更新调整，分别如下：

## 1.历史天气数据近实时更新

全国市区县逐小时历史天气数据已同步更新到2026年4月17日，更新策略由以往的每季度或半年整体更新一次调整为近实时更新，一般ERA5数据源最新数据可获取到当前日期前10天左右，现在海天气象也提供相近时次的最新历史天气数据，所有全国3191个市区县的最新历史天气数据会**每天**同步更新到当前日期约10日前，如今天市2026年4月27日，用户可获取到4月17日的历史天气数据，可通过数据在线下载和API调取的方式同时获取最新历史天气数据，如果不知道最新数据日期，可通过API按月调取数据接口调取最新月份数据。参考代码如下：

```
# -------------------------------------3.调取某月历史天气数据----------------------------------------
# 年月参数，整型，须为2000年1月至今日之间的月份，历史天气数据更新有1个月左右的延迟，所以近1个月的数据可能无法调取
year, month = 2025, 3

# 地名参数，目前收录全国3191个市区县，城市列表请在说明文档中下载查看（https://htqx.cn/meteo/download/resource/cities_list.csv）
city_name = '成都市'

# 构造url，获取该城市该月份内720个时次的历史天气数据
history_monthly_url = f"https://htqx.cn/meteo/api/history/monthly/{city_name}/{year}/{month}/{api_key}"
history_monthly_data = requests.get(history_monthly_url)
print(history_monthly_data.text)
```

在[数据下载页面(https://htqx.cn/meteo/data/history/)](https://htqx.cn/meteo/data/history/)也可下载最新的历史天气数据CSV文件。数据下载方式和API接口没有任何更改。但是统计分析历史天气数据不提供近实时更新频率。

## 2.天气预报数据开放ECMWF原始数据接口

现在提供原始ECMWF未来10日逐小时天气预报数据接口，可以获取IFS 0.1°分辨率最新数值天气预报数据，支持全国地名调取和任意经纬度调取，数据源与ECMWF保持一致，每日更新频率提升至2次，提供每日08时和20时数据，并采用统一API接口调取，确保用户始终获取的是ECMWF最新数据，一般08时数据会在北京时间17时左右更新完毕，20时数据会在北京时间05时更新完毕，目前仅支持通过API获取数据，不支持批量打包下载全国市区县天气预报数据，参考API示例代码如下：

```
# -------------------------------------1.3 通过地名调取ECMWF滚动更新预报----------------------------------------
# 日期年月日参数，整型，须为近30日的日期
# 地名参数，目前收录全国3191个市区县，城市列表请在说明文档中下载查看（https://htqx.cn/meteo/download/resource/cities_list.csv）
city_name = '西安市'

# 构造url，获取该城市该日期起，未来10日逐小时天气预报
forecast_daily_url = f"https://htqx.cn/meteo/api/forecast/ecmwf//location/{city_name}/{api_key}"
forecast_daily_data = requests.get(forecast_daily_url)
print(forecast_daily_data.text)

# -------------------------------------1.4 通过地理坐标调取ECMWF滚动更新预报----------------------------------------
# 地理坐标经纬度参数
lon, lat = 108.940509, 34.617382

# 构造url，获取该坐标最新未来10日逐小时天气预报，如果需要查询确认坐标地名信息，参考第5节
forecast_coords_url = f"https://htqx.cn/meteo/api/forecast/ecmwf/coords/{lon}/{lat}/{api_key}"
forecast_coords_data = requests.get(forecast_coords_url)
print(forecast_coords_data.text)
```

原先的天气预报数据接口和天气预报数据下载方式保持不变，用户可自行选用合适的API调取天气预报数据，由于新的数据接口每日更新频率为2次，时效性更好，后续将逐步采用新的接口提供数据并考虑放弃旧的天气预报接口，目前用户可不必调整代码和数据下载方式，新旧天气预报数据提供的气象要素类型基本一致，预报时长均为未来10日，时间分辨率为逐小时，均支持地名调取和坐标调取。

# 全国城市逐小时空气质量数据集现已更新

海天气象数据信息服务平台（[https://htqx.cn/](https://htqx.cn/)）现已将全国城市逐小时空气质量数据集更新至2026年5月30日，该数据集源自**CNEMC 中国环境监测总站**，目前用户可在网页选取全国333个地级市的空气质量数据下载，空气质量数据参数包括AQI、PM2.5、PM10、SO2、NO2、O3、CO浓度等7项，时间分辨率均为逐小时，时间跨度从2014年至2026年共计13年，格式为CSV，便于直接读取查看和编程处理，数据组织形式为单个城市封装为一个独立CSV文件，共计11万行，索引为时间（每小时），共有7列分别对应7种空气质量参数，气体浓度单位均为微克/立方米，用户既可直接下载数据到本地，也可通过API在线调取数据。

## 一、数据概况

全国城市2014~2026逐小时历史空气质量数据参数如下：

- 名称：全国城市逐小时历史空气质量数据集
- 来源：CNEMC 中国环境监测总站
- 时间分辨率：1小时
- 覆盖范围：中国333个地级市
- 要素种类：7种
- 获取方式：在线下载或API调取
- 单文件大小：5.42 MB
- 数据格式：CSV
- 最新数据日期2026年5月30日23时

## 二、获取方式

### 1.网页直接下载

打开海天气象数据信息服务平台（[https://htqx.cn/](https://htqx.cn/)），点击导航栏数据，选择“空气质量数据”，即可跳转到数据列表，点击右上角“查找城市”，点击需要的城市即可跳转到该城市的空气质量数据列表，然后点击右侧“下载数据”按钮即可完成下载。
点击详情按钮可查看数据详情，点击右上角下载测试数据按钮可下载样例数据。

### 2.API调取

示例URL如下：

```
https://htqx.cn/meteo/api/aqi/yearly/广州市/2025/api_key
```

其中“广州市”和“2025”为可变参数，分别代表需要调取空气质量数据的城市和年份，示例URL返回数据为8760行，代表广州市2025年全年的逐小时空气质量数据，AQI、PM2.5、PM10、SO2、NO2、O3、CO浓度等7项均包含在其中。
